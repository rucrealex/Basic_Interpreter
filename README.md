# Basic_Interpreter
Simple Basic Interpteter writen in JAVA

# Commands:
1. LET ...
2. IF ... THEN GOTO ...
3. PRINT ...
4. Arithmetics operations: +, -, /, *
5. GOTO
6. LIST
7. RUN

# Example basic program listing:
- 10 LET x = 10
- 20 LET y = 20
- 30 LET z = 2*x*x + y
- 40 PRINT z
- 50 IF z > 100 THEN GOTO 100
- 100 PRINT "z more then 100"
- LIST
- RUN
